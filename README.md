## Console_Tetris
Console tetris based on Blam!Games' mini-lib that uses WinApi.

Game Action | Keyboard
------------|----------
Move Left | Left Arrow
Move Right | Right Arrow
Soft Drop | Down Arrow
Gravity | Enter
Rotation | Space bar
